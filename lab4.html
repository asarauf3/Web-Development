Experiment 4: Applying CSS (Internal Method)

Objective: Apply Basic Styling to the HTML Page

This entire page uses **Internal CSS**, defined in the **<style>** tag within the **<head>** of the document. All paragraphs on this page share the same light gray border and padding, as defined in the CSS rules.

The background color of the body is set to a light gray, and the main heading is centered and dark blue. This is the power of CSS to separate presentation from content.
